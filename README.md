# codex-test

## 在庫メモアプリ

ブラウザだけで動く 1 ページ完結の在庫メモアプリです。`index.html` を開くだけで使えます。

### 機能
- 商品名 / 在庫数 / 価格の入力
- 追加した商品の一覧表示
- 各商品の削除
- `localStorage` 保存（リロード後もデータ保持）

## 動作確認手順
1. ブラウザで `index.html` を開きます。
2. 「商品名」「在庫数」「価格」を入力し、「追加する」を押します。
3. 下の在庫一覧に追加されることを確認します。
4. 追加した行の「削除」を押し、一覧から消えることを確認します。
5. いくつか商品を追加した状態でブラウザをリロードし、データが保持されることを確認します。
